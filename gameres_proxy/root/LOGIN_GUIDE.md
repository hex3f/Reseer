# 登录系统使用指南

## 工作原理

### 本地模式
- 前端 → 本地API服务器 (127.0.0.1:8211)
- 任意用户名密码都可以登录
- 数据保存在本地

### 官服模式  
- 前端 → 官服API服务器 (45.125.46.70:8211)
- 需要真实的官服账号
- 数据来自官服

## API适配器自动切换

`official-api-adapter.js` 会自动检测当前模式：

1. 首次请求时，调用 `http://127.0.0.1:8211/api/mode` 检测模式
2. 根据返回的 `local_server_mode` 决定使用哪个API服务器
3. 本地模式 → 使用 `http://127.0.0.1:8211`
4. 官服模式 → 使用 `http://45.125.46.70:8211`

## 测试步骤

1. 访问 http://127.0.0.1:32400/test.html
2. 点击"运行所有测试"
3. 查看日志输出，确认：
   - 当前模式（本地/官服）
   - API服务器地址
   - 登录是否成功

## 登录页面

- `login-official.html` - 完整的Vue登录页面
- `login-vue.html` - 简化的Vue登录页面
- `index.html` - 原始登录页面

所有页面都会根据模式自动选择正确的API服务器。
